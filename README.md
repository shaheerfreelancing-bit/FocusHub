# 🚀 FocusHub

FocusHub is a modern productivity dashboard built to help users stay focused, organized, and productive throughout the day.

## ✨ Features

- 🕒 Live Digital Clock
- 🌦️ Real-time Weather
- 📅 Interactive Calendar
- ✅ Daily Task Manager
- 🍅 Pomodoro Pro Timer
- 📝 Quick Notes
- 🌐 Favorite Websites
- 🎯 Daily Goal Tracker
- 🌙 Dark / Light Theme
- 🔍 Google Search
- ✨ Interactive UI Animations
- 🎨 Glassmorphism Design
- 📱 Fully Responsive Layout

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript
- Vite

## 🎯 Purpose

FocusHub was created to provide an all-in-one workspace where students, developers, creators, and professionals can manage their daily productivity in a clean and modern interface.

## 📸 Preview

A sleek gaming-inspired productivity dashboard with modern UI, glowing effects, animations, weather integration, Pomodoro timer, tasks, notes, and productivity tools.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/shaheerfreelancing-bit/FocusHub.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
